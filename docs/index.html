FAQ · 常见疑问

下载 / 使用 常见问题

是否需要联网 / API Key？

软件本身离线可用：文件库 / 金句库 / 历史 / 导出全部本地完成。只有调用 AI（分析、生成、图标、修图）时才需要联网，需要你在「设置」里填一次 OpenAI 兼容 API Key。未配置 Key 时也可以走 Demo 流程通体演示。

数据存在哪里？我能搬到另一台电脑吗？

素材正文存于 %LOCALAPPDATA%\AIContentStudio\userData 下的磁盘文件库；索引、历史、金句、设置存于浏览器内核的 IndexedDB（ACSPProDB）。进入「设置 → 导出全部数据」即可导出 JSON 备份，到新机器「导入」即可。

API Key 安全吗？

Key 由 Electron safeStorage 加密落盘（DPAPI / Keychain），不会出现在 IndexedDB / 备份 JSON / 历史中。源码里也有「设置 → 一键清空全部数据」的回收站。

未来会有 Mac / Linux 版本吗？

技术上 Electron 跨平台，工具链也都支持；目前优先打磨 Windows 版稳定性，Mac / Linux 包会在主线功能稳定后随主版本一起出。

升级会不会丢老数据？

不会。新版本沿用同名 IndexedDB（ACSPProDB）与磁盘库目录；从「AI Content Studio Pro」改名为「藏经阁」后数据库与文件库目录保持兼容。建议升级前先「设置 → 导出全部数据」做一次备份。

下载链接打不开？

站点本身为静态页，文件托管于 GitHub Releases。如本地未配置仓库地址，按钮可能仍是占位符，点上面的「⚙ 配置仓库地址」填入你自己的 GitHub 用户名/仓库 即可。
