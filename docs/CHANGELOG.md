# 藏经阁 · 更新日志

格式说明：每个版本一段，`Added` / `Changed` / `Fixed` / `Removed` 四类，时间倒序。

---

## v1.0.31 — 2026-05-09

### Added · 文档生成器二期 12 项进阶

- **章节工具**：预览中每个 H2/H3 旁出现 `🔄 重写 / ✂ 删 / ➕ 插入 / ⬆ 上移 / ⬇ 下移`；重写本节会带全文上下文，AI 改写不会破坏其它章节。
- **并行 A/B/C**：一次同时生成 1/2/3 个变体（不同写作风格），结果以 tab 并排展示，**双击 tab 即可采用为最终稿**。
- **生成队列**：`＋加入队列` 累积多个生成任务，`▶ 运行` 依次执行，每个完成都进历史。
- **封面页**：开关式开启后，所有导出会自动在文档前加一页 `# 标题 / 公司 / 保密戳 / 日期`。
- **品牌包**：弹窗设置公司名 / 主色 / 页脚文字 / Logo（base64 嵌入）；自动注入到导出 YAML 元数据。
- **模板 I/O**：一键把所有「生成预设 + 我的模板 + 最近请求」打包成 `.tpl.json`，新机器选文件回灌。
- **批注**：浮条 `💬 批注` → 输入备注 → 预览中黄底高亮；侧栏面板可定位 / 删除。
- **表 → 图**：自动检测 Markdown 表格 → 生成 Mermaid `pie` 或 `xychart-beta bar` 块插入表格下方。
- **AI 自评**：`🎯 自评` 按钮让 AI 给当前输出 1–10 分并附改进建议。
- **版本对比**：`🔀 版本对比` 弹窗，选两个版本看 LCS diff。
- **章节移动**：上移/下移单个章节，无需手工剪切。
- **Bundle 场景矩阵**：导出面板新增 4 套快速勾选预设（客户提案/老板汇报/双语稿/全格式）。

### Added · 文档生成器一期 11 项基础

- **打字机渐显**：长文输出按 600+ 字阈值触发，模拟流式生成体验。
- **目标字数 + 容差**：输入期望字数和 ±%；超出范围有提示。
- **预设 / 我的模板 / 最近 10 条**：所有生成配置可保存复用。
- **Token 预算**：生成前估算 Token 数和成本（输入 + 输出），过大时变红色警告。
- **键盘快捷键**：`Ctrl+Enter` 生成、`Ctrl+S` 存版本、`Ctrl+Shift+E` 切换导出 tab、`Ctrl+K` 聚焦输入。
- **大纲优先**：先出大纲、确认后再展开全文。
- **选中文字浮条**：`改写 / 缩短 / 扩写 / 译中 / 续写 / 挑刺 / ✎自定义`，只对选中段落调 AI。
- **存为金句 / 存进文件库**：一键把输出拆段进金句库，或整篇进文件库。

### Added · 桌面 / 品牌

- 项目改名 **「藏经阁」**：窗口标题、安装目录、桌面快捷方式、应用图标全部更新。
- 新版应用图标：朱砂红底 + 烫金「藏」字方印。
- 侧边栏栏目用书法字（案/卷/鉴/笔/印/绘/句/史/设）替代几何图标。
- 金句库分类卡片加水印汉字 + 朱印章式分类标签。
- 关键词搜索：找到的句子加显眼黄底高亮 + 上下导航控件。

### Added · 官网与发布

- 完整自包含官网（`docs/index.html`）：Hero / 功能九宫格 / 亮点 / 下载（带 SHA256 校验）/ FAQ。
- `docs/releases.json` 数据驱动元数据；GitHub 仓库地址支持页内配置 + `?gh=owner/repo` 参数。
- `scripts/package-portable.ps1`：一键把 `win-unpacked` 打成便携 zip，输出 SHA256。
- `scripts/update-release-meta.js`：构建后自动刷新 `releases.json` 中的版本、大小、SHA256。

### Changed

- `package.json`：NSIS 产物文件名改为 `藏经阁-Setup-${version}.exe`（连字符更适合 URL）。
- 部署脚本默认在 `robocopy` 前 `Stop-Process` 关闭运行实例，避免文件锁。
- 多个 PowerShell 脚本统一加 UTF-8 BOM，修复中文字符串在 PS 5.1 的乱码。

### Fixed

- 生成器 floatBar 未在某些路径定义时崩溃。
- 文件库导入大量条目时进度卡顿。
- 桌面快捷方式残留旧名「AI Content Studio Pro.lnk」，安装新版本时不自动清理。
